
# Trent Ltd – Financial Statement Model & DCF Valuation

This repository contains a fully integrated 3-statement financial model and a Discounted Cash Flow (DCF) valuation for Trent Ltd (Tata Group).

The project focuses on clean accounting linkages, conservative operating assumptions, and a valuation framework aligned with equity research standards.

---

## Model Components

### 1. Financial Statement Model (FSM)
- Forecast period: FY25–FY27
- Fully linked Income Statement, Balance Sheet, and Cash Flow Statement
- Store expansion–driven revenue growth assumptions
- Explicit modeling of working capital and capital expenditure
- Clean handling of circularity

### 2. Discounted Cash Flow (DCF)
- Free Cash Flow to Firm (FCFF)–based valuation
- WACC derived using CAPM assumptions
- Terminal value calculated using the Gordon Growth method
- Sensitivity analysis on WACC and terminal growth
- Equity value and implied share price estimation
